## Introduction

This program generates a sentence including an exclamation/question mark between each characters.

## Implementations

- C
- Haskell
- Python

## Build

```
make
```

## Usage

```
./nsy < TEXT_FILE_NAME

where
   TEXT_FILE_NAME = the file name to be input-redirected.
```
